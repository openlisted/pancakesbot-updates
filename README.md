## 🛠️ 功能特性 (Features)

### 1. 自动化做市 (Market Making)
*   **多模式支持**：
    *   **顺序模式**：子钱包轮流交易，模拟真实用户增长。
    *   **随机模式**：交易时间、金额、钱包选择全随机，规避女巫检测。
*   **参数微调**：
    *   自定义价格区间（高抛低吸）。
    *   自定义单笔交易金额范围。
    *   自定义交易间隔时间。

### 2. 便捷的钱包管理 (Wallet Manager)
*   **批量生成**：一键生成指定数量的新钱包。
*   **批量归集**：一键将所有子钱包的代币/BNB 归集到主钱包。
*   **批量分发**：一键向子钱包分发 Gas 费或代币。
*   **权益共享**：主账户开通 VIP，旗下绑定的子账户**自动继承** VIP 权益，无需重复付费。
